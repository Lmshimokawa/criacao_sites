# Copy — Páginas para Experiências Premium (v1.0)

## Objetivo

Criar copy completo para sites de experiências/serviços premium, com foco em conexão emocional, tom conversacional e conversão sem pressão.

## Quando Usar

- Após definir wireframes/layout
- Quando o negócio vende experiências (não apenas produtos)
- Para público que toma decisões emocionais
- Quando autenticidade é mais importante que formalidade

## Inputs Necessários

- **Proposta de valor** (da Fase A)
- **Tom de voz da marca** (íntimo, autêntico, acolhedor, etc.)
- **Wireframes** (estrutura das páginas)
- **Público-alvo** (perfil comportamental)
- **Ofertas** (produtos/serviços com detalhes)
- **CTAs principais** (por página)
- **FAQ** (perguntas frequentes)

## Prompt

```
Você é um copywriter especializado em marcas de experiências premium, 
com foco em conexão emocional e conversão sem pressão.

CONTEXTO DA MARCA:
- Nome: [NOME]
- Proposta de valor: [PROPOSTA]
- Tom de voz: [TOM - ex: íntimo, autêntico, acolhedor, poético mas acessível]
- Público-alvo: [PERFIL]
- Ofertas: [LISTA DE OFERTAS]

ESTRUTURA DAS PÁGINAS:
[WIREFRAMES OU LISTA DE SEÇÕES]

TAREFA:
Crie copy completo para cada página seguindo estes princípios:

1. TOM DE VOZ
   - Conversacional (como falar com uma amiga)
   - Autêntico (sem jargões corporativos)
   - Acolhedor (convite, não pressão)
   - Confiante mas não arrogante

2. ESTRUTURA DE COPY
   
   HEADLINES:
   - Curtos e impactantes
   - Benefício ou emoção, não feature
   - Específicos ao contexto
   
   SUBHEADLINES:
   - Expandem o headline
   - Adicionam contexto necessário
   - Mantêm o ritmo
   
   BODY COPY:
   - Parágrafos curtos (2-3 linhas)
   - Uma ideia por parágrafo
   - Progressão lógica
   
   CTAs:
   - Ação clara
   - Benefício implícito
   - Sem urgência falsa

3. PRINCÍPIOS
   - Emocional primeiro, informativo depois
   - Benefício, não feature
   - Específico, não genérico
   - Mostrar, não contar
   - Escassez implícita (não agressiva)

4. EVITAR
   - Superlativos vazios ("o melhor", "incrível")
   - Urgência falsa ("últimas vagas!")
   - Premium/luxo explícito
   - Parágrafos longos
   - Linguagem corporativa

5. FORMATO DE SAÍDA
   Para cada página:
   
   ## [NOME DA PÁGINA]
   
   ### [SEÇÃO]
   
   **Headline:**
   [texto]
   
   **Subheadline:**
   [texto]
   
   **Body:**
   [texto]
   
   **CTA:**
   [texto]

6. ELEMENTOS ESPECIAIS
   - FAQ: Perguntas naturais, respostas conversacionais
   - Depoimentos: Estrutura com emoção + benefício + nome
   - Microcopy: Orientações curtas e amigáveis
   - Meta tags: SEO-friendly mas naturais

OUTPUT ESPERADO:
- Copy completo para cada página
- Headlines, subheadlines, body copy, CTAs
- FAQ com respostas
- Meta tags para SEO
- Tom consistente em todas as páginas
```

## Output Esperado

### Critérios de Sucesso

1. **Tom consistente**
   - Mesma voz em todas as páginas
   - Autêntico e conversacional
   - Sem quebras de personalidade

2. **Conversão facilitada**
   - CTAs claros e convidativos
   - Benefícios evidentes
   - Objeções respondidas

3. **Experiência fluida**
   - Progressão lógica
   - Informação encontrável
   - Sem confusão

4. **SEO-friendly**
   - Meta tags otimizadas
   - Keywords naturais no texto
   - Estrutura clara
