# Strategy — Proposta de Valor (v1.0)

## Objetivo

Definir a proposta de valor de um negócio, indo além da descrição técnica do serviço ou produto. Identificar a transformação emocional, a memória criada e o valor percebido pelo cliente.

## Quando Usar

- Início de um projeto de site (Fase A — Arquitetura de Valor, Lição A1)
- Reposicionamento de marca
- Criação de copy institucional
- Definição de estratégia de comunicação

## Inputs Necessários

- **Nome do negócio**
- **Serviço/produto oferecido** (descrição técnica)
- **Público-alvo** (quem compra)
- **Contexto de uso** (onde/como o serviço é entregue)
- **Diferenciação** (o que torna único)
- **Objetivo do site** (institucional, venda, autoridade, etc.)

## Prompt

```
Você é um estrategista de marca especializado em definir propostas de valor que vão além da descrição técnica.

CONTEXTO DO NEGÓCIO:
- Nome: [NOME DO NEGÓCIO]
- Serviço/Produto: [DESCRIÇÃO TÉCNICA]
- Público-alvo: [QUEM COMPRA]
- Contexto de uso: [ONDE/COMO É ENTREGUE]
- Diferenciação: [O QUE TORNA ÚNICO]
- Objetivo do site: [INSTITUCIONAL/VENDA/AUTORIDADE]

TAREFA:
Defina a proposta de valor deste negócio seguindo estas etapas:

1. ANÁLISE DO VALOR REAL
   - O que o cliente compra ALÉM do serviço técnico?
   - Que problema emocional é resolvido?
   - Que memória ou identidade é reforçada?
   - Qual a transformação percebida (antes → durante → depois)?

2. COMPONENTES DA PROPOSTA
   - Valor funcional: O que o serviço faz
   - Valor emocional: Como o cliente se sente
   - Valor social: Como o cliente é visto
   - Valor de memória: O que fica depois

3. DIFERENCIAÇÃO
   - O que este negócio NÃO é (para evitar confusão)
   - O que este negócio É (posicionamento claro)
   - Território simbólico ocupado

4. ENTREGÁVEIS
   Crie os seguintes artefatos:

   a) POSICIONAMENTO 1-LINHA
      - Frase que define o negócio em uma linha
      - Clara, memorável, diferenciada
      - Transmite valor emocional

   b) PARÁGRAFO INSTITUCIONAL PREMIUM
      - Texto de 3-4 frases para página "Sobre" ou hero da Home
      - Sofisticado mas acessível
      - Conecta emocionalmente
      - Transmite proposta de valor completa

   c) TAGLINES (3 opções)
      - Variadas em tom (emocional, funcional, social)
      - Memoráveis
      - Alinhadas com proposta de valor
      - Indique uso recomendado para cada uma

   d) OBJEÇÕES E RESPOSTAS CURTAS
      - Liste 5 objeções comuns do público-alvo
      - Para cada objeção, forneça resposta concisa (1-2 frases)
      - Respostas devem reforçar proposta de valor
      - Tom respeitoso, não defensivo

RESTRIÇÕES:
- Não simplifique conceitos importantes
- Profundidade é obrigatória em propostas de valor
- Evite clichês genéricos ("transformamos vidas")
- Seja autêntico e específico
- Foque em valor percebido, não apenas funcionalidades

OUTPUT ESPERADO:
- Análise completa da proposta de valor
- Posicionamento 1-linha
- Parágrafo institucional premium
- 3 taglines com uso recomendado
- 5 objeções com respostas curtas
- Justificativa para cada escolha
```

## Output Esperado

### Critérios de Sucesso

1. **Proposta de valor clara e diferenciada**
   - Não é genérica
   - Vai além do serviço técnico
   - Transmite valor emocional

2. **Posicionamento 1-linha**
   - Memorável
   - Diferencia de competidores
   - Transmite valor emocional

3. **Parágrafo institucional**
   - Sofisticado mas acessível
   - Conecta emocionalmente
   - Completo (cobre todos os componentes)

4. **Taglines**
   - 3 opções variadas
   - Uso recomendado claro
   - Alinhadas com proposta

5. **Objeções e respostas**
   - Objeções reais do público-alvo
   - Respostas que reforçam proposta
   - Tom respeitoso

---

## Notas de Uso

- Este prompt foi criado especificamente para negócios de serviços premium e experiências
- Para produtos físicos, ajustar foco em "valor de memória" e "experiência"
- Para B2B, focar mais em valor funcional e ROI, mas não ignorar emocional
- Sempre validar proposta de valor com público-alvo antes de usar em produção
